\begin{abstract}
Al momento de desarrollar sistemas web utilizando lenguajes que corren en la Java Virtual
Machine (\emph{JVM}), es a menudo necesario descargar una serie de bibliotecas y archivos
desde diversos sitios de Internet para agregarlos a nuestro proyecto. Estos archivos son
nuestras \emph{dependencias} para la parte visible (interfaz gráfica o \emph{front-end})
de nuestro sitio. Esta tarea suele llevar tiempo, ser tediosa y propensa a errores.
Actualmente no hay herramientas existentes que solucionen el problema  utilizando
solamente Java o algún lenguaje que corre en la \emph{JVM}.
Este trabajo propone una solución al problema mediante el desarrollo de un manejador de
dependencias hecho completamente en \emph{Scala}, capaz de descargar automáticamente todos
estos archivos de forma automática.
\end{abstract}
